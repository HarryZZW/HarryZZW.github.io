# Harrison Zhang personal website

Static GitHub Pages website for Harrison Zhang's academic job-market profile.

## Contents

- `index.html` — main website
- `styles.css` — all styling; no external CSS or scripts
- `assets/img/profile.jpg` — profile photo used on the homepage
- `assets/docs/CV_Zhengwei_Zhang_public.pdf` — public CV PDF
- `404.html` — simple fallback page

## Publishing on GitHub Pages

Upload the contents of this folder to the root of the `USERNAME.github.io` repository so `index.html` appears at the top level.

The package is intentionally static: no contact form, no analytics, no external fonts, and no JavaScript.
