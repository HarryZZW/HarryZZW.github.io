# Security checklist

- Use a strong unique GitHub password.
- Enable two-factor authentication or passkeys on GitHub.
- Keep this repository public-facing and static.
- Do not upload private drafts, raw data, API keys, `.env` files, or private PDFs.
- Only the public CV PDF is included in this package.
- The job market paper PDF and research statement PDF are not included.
- The site uses no contact form, database, tracking scripts, or third-party JavaScript.
